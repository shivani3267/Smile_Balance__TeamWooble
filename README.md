# **Project Overview**

In today’s fast-paced digital world, people are experiencing increased stress, loneliness, and emotional fatigue. At the same time, many individuals are searching for simple ways to earn small amounts of money online. Most earning platforms require skills, time, or investment, making them inaccessible to many people. There is a strong need for a system that improves mental well-being while also offering micro-income opportunities.

# **Project Description:**

SmileBalance is a web-based platform that allows users to earn money by simply smiling. Users register securely through OTP-based phone number authentication to ensure one account per person. The platform uses AI-powered face and smile detection to verify that each uploaded selfie contains a real and genuine smile. Once verified, users receive reward credits for every smile. These credits can later be converted into real money after reaching a minimum withdrawal threshold. To prevent misuse, users are limited to two smile submissions per day with a six-hour gap between them. A personal dashboard tracks smile history, earnings, and streaks, motivating users to stay consistent. SmileForHope combines technology, positivity, and financial motivation into one impactful solution.

# **Project Features**
<ol>

<li>AI-based face and smile verification</li>

<li>Reward credits with real-money conversion</li>

<li>Two smiles per day with a 6-hour gap</li>

<li>User dashboard with smile count, earnings, and streaks</li>

<li>Fraud prevention through face matching and image validation</li>

<li>Brand and NGO-sponsored reward model</li>
</ol>

## Tech Stack

**Frontend:** React ,Tailwind CSS, Javascript

**Backend:** Node.js, Express.js

**Database:** MongoDB

**Authentication:** JWT Authentication

**AI / Image Processing:**  Face-API.js

## Installation

### Frontend

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shivani3267/Smile_Balance__TeamWooble.git

2. **Install frontend dependencies:**
   ```bash
   cd frontend
   npm install

3. **Start the server:**
   ```bash
   npm start

### Backend 

1. **Install backend dependencies:**
   ```bash
   cd backend
   npm install

2. **Create Environment File**
   ```bash
   cd Backend
   touch .env

3. **Edit Environment Variables**
   To run this project, you will need to add your mongodb config of database like this to your .env file and secret key for jwt
   `JWT_SECRET`
   `MONGO_URI`

4. **Run the Backend Server**
   ```bash
   node server.js
